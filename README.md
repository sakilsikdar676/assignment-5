*    - 1️⃣ What is the difference between var, let, and const?
  Ans: var, let and const are used to declare variables in Javascript. The main difference is their scope and how they can be updated. var is function-scoped and can be reduclared and updates. let is block-scoped and can be updated but can't be redeclared in the same scope. const is also block-scoped but its values can't be reassigned after it is declared.

    - 2️⃣ What is the spread operator (...)?
  Ans: The spread operator(...) is a javascript feature used to expand or spread the elemeents of an array or object into individual elements. It is commonly used to copy, merge or pass multiple values easily.
  
    - 3️⃣ What is the difference between map(), filter(), and forEach()?
  Ans: map(), filter() and forEach() are arry methods used to work with array elements. map() is used to transform each of an array and returns a new array. filter() is use to select elements based on a condition and also returns a new array. forEach() is used to loop through each element of an array to perform an action, but it does not return a new array.

    - 4️⃣ What is an arrow function?
  Ans: An arrow function is a shorter and more modern way to write functions in Javascript. It provides a concise syntax and is often used for simple functions and callbacks.

    - 5️⃣ What are template literals?
  Ans: Template literals are a feature in javascript used to creat strings more easily. They allow embedding variables or variables or expression inside a string and also support multi-line text formatting.*
